# 🐕 Rex & os Ossos — Aprenda Algoritmos

Jogo educativo de lógica de programação com 40 fases progressivas, criado para ensinar conceitos de algoritmos de forma visual e interativa.

## 🎯 Como jogar

1. Monte um programa usando os **comandos disponíveis**
2. Clique em **▶ Executar** para ver o Rex se mover
3. O objetivo é levar o Rex até o osso 🦴
4. Cada fase ensina um novo conceito de programação!

## 📚 Conceitos ensinados (40 fases)

| Fases | Conceito |
|-------|----------|
| 1–3   | Sequência e planejamento |
| 4, 7  | Laços: `repetir(N)` e `enquanto` |
| 5–6   | Condicionais: `se` / `senão` |
| 8–10  | Combinando lógica |
| 11–20 | Rota, otimização, variáveis |
| 21–30 | Padrões, busca, labirinto |
| 31–40 | Perigo, bônus, escalabilidade, desafio final |

## 🚀 Como hospedar no GitHub Pages

1. Crie um repositório no GitHub (pode ser público)
2. Faça upload do arquivo `index.html` e `README.md`
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `main` branch e pasta `/ (root)`
5. Clique em **Save**
6. Acesse em: `https://SEU-USUARIO.github.io/NOME-DO-REPO`

> O site fica disponível em alguns minutos após ativar o Pages.

## 🎨 Comandos do jogo

```
mover()               → Move o Rex 1 casa na direção atual
virar_direita()       → Vira 90° para a direita
virar_esquerda()      → Vira 90° para a esquerda
repetir(N)            → Repete mover() N vezes
se(tem_parede_frente) → Executa próximo cmd se houver parede
se(tem_osso_frente)   → Executa próximo cmd se osso à frente
se(tem_perigo_frente) → Executa próximo cmd se perigo à frente
se(borda_frente)      → Executa próximo cmd se borda do mapa
senao                 → Par do SE: executa se condição falsa
enquanto(sem_osso)    → Loop: repete até encontrar o osso
```
